### 🚀 Sobre o projeto
O System Resume simula uma ferramenta simples de triagem de curriculos, muito utilizada em processos seletivos.
A aplicação lê arquivos PDF, extrai o texto e gera um resumo automático com base nas informações encontradas.


### ✨ Funcionalidades
+ Leitura de currículos em PDF
+ Extração automática de texto
+ Geração de resumo de perfil
+ Execução via terminal
+ Código simples e organizado


### 🛠️ Tecnologias utilizadas
+ Python
+ PyPDF2
+ JSON
+ Programação estruturada

### ⚙️ Como executar o projeto
### 1️⃣Clone o repositório
```bash
git clone https://github.com/seu-usuario/resume-reader.git
cd resume-reader
```
### 2️⃣Instale as dependências
```bash
pip install PyPDF2
```
### 3️⃣Adicione um currículo
Coloque um arquivo PDF dentro da pasta do projeto
Exemplo: curriculo.pdf

### 4️⃣Execute a aplicação
```bash
python main.py
```
